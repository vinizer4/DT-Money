# :computer: How to run

```bash
# Clone Repository
$ git https://github.com/vinizer4/DT-Money.git
```

```bash
# Install Dependencies
$ npm i --force

# Run Aplication
$ npm run dev
$ npm run dev:server
```
Go to http://localhost:3000/ to see the result.
